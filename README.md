# 8-Bit Game School

Learn **HTML, CSS, and JavaScript** by building simple retro-style browser games.

This project contains beginner-friendly games designed to teach the core concepts of web development through interactive examples.

---

#  Games Included

##  Game 1 — Pixel Clicker

Beginner level game where an alien appears randomly on screen.

Your goal is to **click the alien before it disappears**.

### What you learn

* CSS animations
* JavaScript click events
* Random positioning with `Math.random()`
* Score counters
* Timers and countdowns

### How to play

1. Start the game
2. Click the alien when it appears
3. Score as many points as possible before the timer runs out

---

## 🚀 Game 2 — Dodge Blaster

Intermediate level arcade-style game where you control a spaceship.

Your goal is to **dodge falling meteors as long as possible**.

### What you learn

* Keyboard input (`ArrowLeft`, `ArrowRight`)
* Game loops using `setInterval`
* Collision detection
* Dynamic element creation with JavaScript
* Increasing difficulty over time

### Controls

```
←  Move Left
→  Move Right
A  Move Left
D  Move Right
```

# How to Run the Project

## Option 1 — Open directly

Simply open **index.html** in your browser.

```
double-click index.html
```

---

## Option 2 — Using VS Code (recommended)

1. Install **Visual Studio Code**
2. Install the **Live Server** extension
3. Open the project folder
4. Right click **index.html**
5. Click **Open with Live Server**

This allows automatic refresh when you edit the files.
