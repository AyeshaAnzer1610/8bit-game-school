🎮 8-BIT GAME SCHOOL

▶ Learn HTML + CSS + JS by building real games ◀

📁 STEP 1 — SETTING UP VS CODE

What you need:

1. Download Visual Studio Code → code.visualstudio.com
2. Install the "Live Server" extension (by Ritwick Dey)
3. Create a folder on your Desktop called my-games
4. Open that folder in VS Code: File → Open Folder
5. Create a new file: game1.html
6. Right-click the file → "Open with Live Server"

💡 Tip: Live Server auto-refreshes your browser every time you save!
  • 1 Every HTML file starts with <!DOCTYPE html> — this tells the browser "this is a webpage".
  • 2 Put all your CSS styles inside <style> ... </style> in the <head> section.
  • 3 Put all your JavaScript inside <script> ... </script> just before the closing </body> tag.
  • 4 Save your file with Ctrl+S (Windows) or Cmd+S (Mac) — your browser will update automatically!

📄 STEP 2 — YOUR STARTER TEMPLATE

Copy this into EVERY new game file:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My Game</title>
    <style>
      /* Your CSS goes here */
    </style>
  </head>
  <body>
    <!-- Your HTML goes here -->
    <script>
      // Your JavaScript goes here
    </script>
  </body>
</html>
